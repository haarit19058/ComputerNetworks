## How to Run the Code

1. Install dependencies:

   ```bash
   pip install dpkt
   ```

2. Start the server:

   ```bash
   python server.py
   ```

3. Run the client:

   ```bash
   python client.py
   ```

* DNS query results will be stored in **`dns_log.csv`**.
* The detailed report for this assignment is available in **`A1_report.pdf`**.
